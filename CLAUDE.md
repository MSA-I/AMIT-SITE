# CLAUDE.md — Amit Bar Interior Design

Project constitution. Keep it short; update it when these rules change.

## What this is
Bilingual (Hebrew RTL default / English LTR) marketing + portfolio site for **Amit Bar**, a boutique
interior-design studio. Single React SPA, multi-page via React Router. Content and 180 project photos
were migrated from the original WordPress site (`abdesigner.co.il`).

## Stack
- React 19 + TypeScript (strict) + Vite 6
- Tailwind CSS v4 (via `@tailwindcss/vite`, tokens in `src/index.css` `@theme`) — NOT the Play CDN
- Framer Motion (animation), Lucide (icons), React Router v7
- No backend. Contact form hands off to WhatsApp (`wa.me`) + mailto.

## Commands
- `npm run dev` — local dev (port 3000)
- `npm run build` — typecheck + production build
- `npm run preview` — serve the build
- `npm run scrape` — re-download + optimize project photos from the live site → `public/projects/` + `src/data/projects.json`

## Structure
- `src/i18n/` — `he.ts` (canonical), `en.ts` (mirror, typed to `Dict`), `context.tsx` (`LanguageProvider`, `useI18n`). All copy lives here. Add a key to `he.ts` first, then `en.ts`.
- `src/data/projects.ts` — typed projects from `projects.json` (generated by the scraper). Use `title()`, `brief()`, `getProject()`, `usedCategories()`.
- `src/components/ui.tsx` — shared primitives: `Container`, `Section`, `Heading`, `Eyebrow`, `Rule`, `Reveal`, `LangLink`, `btnSolid/btnGhost/btnCopper`. Reuse these; do not re-invent.
- `src/components/` — `Layout`, `Navbar`, `Footer`, `StickyContact`, `Seo`, `ProjectCard`, `ContactForm`, `Lightbox`.
- `src/components/sections/` — page sections (zero-prop, read context/data).
- `src/pages/` — routed pages; mostly compose sections.
- `src/router.tsx` — routes under `/:lang`; `/` → `/he`.

## Design system (LOCKED — Architectural Mono + Accent)
- Light theme only. Tokens: `bg` `#F8F8F8`, `surface` `#fff`, `ink` `#232323`, `muted` `#6B6B6B`, `line` `#E4E1DC`, `copper` `#B87333`.
- One accent: **copper**. Never add a second accent. Body text is `text-ink`; copper is for large type / hairlines / small UI only (it fails AA at body size).
- Shape: sharp (`rounded-none`/`rounded-sm`); pills only via the `btn*` helpers. Hairlines (`border-line`) over heavy shadows.
- Fonts: `Jost` (Latin display) + `Archivo` (Latin body) + `Heebo` (Hebrew). Hebrew gets looser leading.
- Motion ceiling: **2-3/10**. Use `Reveal` for scroll fades. Custom motion must respect `useReducedMotion()`. Never use `window.addEventListener('scroll')` — use Framer `useScroll`/`useInView`.

## RTL / i18n rules
- Use Tailwind **logical** properties only: `ps-/pe-/ms-/me-/start-/end-/text-start/text-end`. Never `pl/pr/ml/mr/left/right`.
- Get language from `useI18n()`. `LanguageProvider` sets `document.dir`/`lang`.
- Links go through `LangLink` or `localePath(lang, path)` so the `/he|/en` prefix is preserved.

## Anti-slop (enforced)
- **Zero em-dash (—) / en-dash (–)** anywhere visible. Hyphen only.
- Real photos only (from `/projects/...`). No fake screenshots, no decorative hand-drawn SVG, no stock.
- No fabricated stats, no version labels, no "scroll" cues, no decorative status-dot spam. Max ~1 eyebrow per section.
